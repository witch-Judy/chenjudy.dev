An Onboarding v1 prototype designed for ILLUCIO's interactive exploration web format.

Instead of a traditional feature carousel, it lets users experience the journey from "can't understand" to "illuminated" firsthand — first feeling the opacity of a dense text, then seeing how ILLUCIO lights it up.
